# Seguridad

## Reporte

Si encuentras una vulnerabilidad, abre un issue o contacta con la persona mantenedora.

## Alcance

SoberTrack no tiene backend ni autenticación. Los riesgos principales están relacionados con datos locales, integridad de la PWA, service worker y mensajes de seguridad preventiva.

## Importante

La app no debe usarse para decidir si una persona puede conducir o realizar actividades de riesgo.
