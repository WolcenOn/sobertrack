# Contribuir

Gracias por mejorar SoberTrack.

## Principios

- Mantener la app sin frameworks externos.
- Priorizar accesibilidad, claridad y rendimiento offline.
- No presentar la estimación BAC como medición real.
- Mantener avisos preventivos visibles.
- Evitar almacenar o enviar datos sensibles fuera del navegador.

## Desarrollo

No hay proceso de build. Sirve el directorio con un servidor local:

```bash
python3 -m http.server 8080
```

## Pull requests

Incluye una descripción clara del cambio, pruebas manuales realizadas y cualquier impacto en privacidad, seguridad o exactitud del modelo.
